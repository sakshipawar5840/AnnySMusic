# 🎵 Anny's Music - Official Artist Website

## 🚀 About Project
Anny's Music ek official artist portfolio website hai jaha music, albums, aur artist information showcase ki jaati hai. Ye website fans ko artist ke latest updates aur music se connect karti hai.

---

## 🖥️ Project Overview
Is project ka purpose ek modern aur attractive music website banana hai jisme artist ka profile, songs, aur contact details display ho.
